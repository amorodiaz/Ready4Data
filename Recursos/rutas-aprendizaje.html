Analista de Datos
Para quienes parten desde cero o con Excel básico y quieren analizar datos y crear dashboards.
⏱ Dedicación estimada: 4–6 meses a razón de 1–2 horas/día
📍 Fase 1 — Fundamentos (mes 1–2)
  • 1
    Excel intermedio-avanzadoTablas dinámicas, BUSCARV, Power Query básico, gráficos
  • 2
    SQL desde ceroSELECT, WHERE, GROUP BY, ORDER BY, funciones básicas
  • 3
    Conceptos de datosQué es una base de datos, tipos de datos, modelo relacional
📍 Fase 2 — Herramientas BI (mes 2–4)
  • 4
    SQL intermedioJOINs, subconsultas, CTEs, funciones de ventana básicas
  • 5
    Power BI fundamentalsConectar datos, modelo de datos básico, visualizaciones, DAX básico
  • 6
    Estadística descriptivaMedia, mediana, moda, varianza, percentiles, correlación básica
📍 Fase 3 — Portfolio y carrera (mes 4–6)
  • 7
    Proyecto 1: análisis de negocioDataset real (e-commerce, RRHH, finanzas) + dashboard en Power BI
  • 8
    Proyecto 2: sector de tu interésDatos del sector donde quieras trabajar
  • 9
    CV, LinkedIn y entrevistasCómo presentar tu portfolio sin experiencia formal
ExcelSQLPower BITableauEstadística básica

Ruta 2 · Nivel 2 → 4
Data Engineer
Para analistas con SQL sólido que quieren construir pipelines y trabajar con datos a escala.
⏱ Dedicación estimada: 6–9 meses a razón de 1–2 horas/día
📍 Fase 1 — SQL avanzado + Python (mes 1–3)
  • 1
    SQL avanzadoWindow functions, CTEs recursivos, optimización, índices
  • 2
    Python para datosPandas, NumPy, manejo de archivos CSV/JSON, APIs básicas
  • 3
    Fundamentos de sistemasTerminal, Git, Docker básico, conceptos de redes
📍 Fase 2 — Plataformas cloud y dbt (mes 3–6)
  • 4
    Cloud Data WarehouseBigQuery o Snowflake: cargar datos, optimizar queries, costes
  • 5
    dbt (data build tool)Transformaciones en SQL con dbt, tests, documentación, lineage
  • 6
    OrquestaciónApache Airflow: DAGs básicos, scheduling, monitorización
📍 Fase 3 — Proyecto end-to-end (mes 6–9)
  • 7
    Pipeline completoIngestión de API → BigQuery → dbt → dashboard
  • 8
    Kafka / Streaming básicoConceptos de procesamiento en tiempo real
PythonSQL avanzadodbtAirflowBigQueryDocker
